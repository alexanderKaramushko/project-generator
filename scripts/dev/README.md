# Dev-скрипты project-generator

## Цель

Упростить разработку и локальное тестирование пакетов. <br/>
Автоматизировать рутинные процессы. <br/>

## Скрипты

| Скрипт            | Назначение |
| -------------     | ------------- |
| link              | Заменяет подключение workspace-пакета из npm на локальный symlink пакета  |
